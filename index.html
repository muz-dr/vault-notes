Connect Your Google Drive
Each person connects their own Google account — your Drive, your data, your control. Follow the steps below.
Step 1 — Create a Google Cloud Project
1Open console.cloud.google.com in a new tab (sign in with your Google account)
2Click the project dropdown at the top → New Project
3Name it anything e.g. my-vault-notes → click Create
4Make sure the new project is selected in the dropdown
Step 2 — Enable the Google Drive API
1In the left sidebar click APIs & Services → Library
2Search for Google Drive API
3Click the result → click the blue Enable button
4Wait for it to enable (about 10 seconds), then continue
💡 It's free — Google Drive API has no cost for personal use
Step 3 — Create OAuth 2.0 Credentials
1Go to APIs & Services → Credentials
2Click + Create Credentials → OAuth client ID
3If prompted, click Configure Consent Screen → choose External → fill in App name (e.g. "Vault Notes") and your email → Save
4Back in Credentials: Application type → Web application
5Under Authorised JavaScript origins add:
https://YOUR-USERNAME.github.io
6Under Authorised redirect URIs add the exact same URL including the page path:
https://YOUR-USERNAME.github.io/vault-notes/index.html
7Click Create → copy the Client ID (ends in .apps.googleusercontent.com)
💡 Both the origin AND the redirect URI must be added. The redirect URI must match your URL exactly — no trailing slash.
Step 4 — Paste Your Client ID
1Paste the Client ID you copied from Google Cloud below
2Tap Save & Connect — it will be stored only on this device
3Then use Save to Google Drive from the menu to back up
Looks like: 123456789-abc123.apps.googleusercontent.com
